# BarleyOne Insights

A static HTML application hosted on Vercel.

## Project Structure

```
project/
├── public/
│   └── index.html       # Main application file
├── vercel.json          # Vercel deployment configuration
├── package.json         # Project metadata
├── .gitignore           # Git ignore rules
└── README.md            # This file
```

## Local Development

To run locally:
```bash
npm run dev
```

The application will open in your browser at `http://localhost:8080`

## Deployment

The application is configured to deploy on Vercel as a static site.
